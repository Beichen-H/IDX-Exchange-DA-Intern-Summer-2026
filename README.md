# IDX Exchange - Real Estate Analytics Platform (Summer 2026)

This repository contains the full end-to-end data pipelines and analytics scripts developed during the IDX Exchange Data Analyst Internship Program. 

##  Tech Stack & Key Concepts
* **Data Processing & ETL:** Python, Pandas, Multi-file dataset management.
* **Statistical Data Cleaning:** Interquartile Range (IQR) outlier detection, date consistency verification, and missing value analysis.
* **Feature Engineering:** Calculation of Price Ratios, PPSF (Price Per Sq Ft), and time-to-sell indicators (Days on Market).

##  Data Privacy Notice
*All datasets used in this program are confidential and sourced from live MLS transaction records. No raw CSV data or source data files are stored in this repository to comply with privacy policies.*
